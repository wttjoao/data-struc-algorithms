# Contains Duplicate

Given an integer array `nums`, return `true` if any value appears more than once in the array, otherwise return `false`.

**Example 1:**

```
Input: nums = [1, 2, 3, 3]
Output: true
```

**Example 2:**

```
Input: nums = [1, 2, 3, 4]
Output: false
```


## Results

### Java

**Time Complexity:** O(n)

**Space Complexity:** O(n)

**Runtime:** 76ms

**Memory:** 62.4MB

### Python

**Time Complexity:** O(n<sup>2</sup>)

**Space Complexity:** O(n)

**Runtime:** 46ms

**Memory:** 7.8MB
